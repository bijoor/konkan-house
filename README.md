# Konkan House - Blender 3D Generator

A Python-based system for procedurally generating 3D house models in Blender with support for sloping roofs, multiple floors, and web deployment.

## üåü Features

- **Procedural Generation**: Define your house layout in Python configuration
- **Multi-floor Support**: Ground floor and multiple upper floors
- **Sloping Walls**: Support for gable roofs with sloping wall tops
- **Doors & Windows**: Automatic boolean operations for openings
- **Materials**: Laterite walls, floor slabs, and customizable materials
- **Web Export**: One-click export to interactive 3D web viewer
- **GitHub Pages Ready**: Auto-generates `docs` folder for deployment

## üöÄ Quick Start

### Prerequisites

- Blender 3.0 or higher
- Python 3.9+ (bundled with Blender)

### Usage

1. **Open Blender**
2. **Load the script**: Open `konkan_house_config.py` in Blender's Text Editor
3. **Run the script**: Click "Run Script" or press Alt+P
4. **Export for web** (optional): Uncomment `export_to_web()` in the config file

## üèóÔ∏è Configuration

Edit `house_config.py` to customize your house:

```python
'floors': [
    {
        'floor_number': 0,
        'objects': [
            {
                'type': 'room',
                'name': 'Living Room',
                'x': 0, 'y': 0,
                'width': 200, 'height': 150,
                'walls': ['north', 'south', 'east', 'west']
            },
            {
                'type': 'wall',  # For sloping walls
                'name': 'Gable_East',
                'height': 100,
                'height_end': 150,  # Creates slope
            },
            {
                'type': 'pillar',
                'x': 100, 'y': 100,
                'height': 100
            }
        ]
    }
]
```

## üåê GitHub Pages Deployment

### Setup (One-time)

The `docs/` folder already contains all static files:
- ‚úÖ `index.html` - Interactive 3D viewer
- ‚úÖ `README.md` - Documentation
- ‚ö†Ô∏è `konkan_house.glb` - Model file (generated by Python, excluded from git)

### Workflow

1. **Build and export your model**:
   ```python
   # Run the script in Blender
   export_to_web()  # Creates docs/konkan_house.glb
   ```

2. **Test locally**:
   - Open `docs/index.html` in your web browser
   - The 3D model should load and be interactive

3. **Commit to GitHub** (first time only):
   ```bash
   git add docs/index.html docs/README.md
   git commit -m "Add 3D model viewer"
   git push
   ```
   Note: The `.glb` file is excluded from git (see `.gitignore`)

4. **Enable GitHub Pages**:
   - Go to repository Settings > Pages
   - Source: "Deploy from a branch"
   - Branch: `main`, Folder: `/docs`
   - Save

5. **Access your site**:
   `https://YOUR_USERNAME.github.io/YOUR_REPO/`

### Updating the Model

Simply run the script again:
```python
export_to_web()  # Regenerates only the .glb file
```

The HTML viewer will automatically load the updated model when you refresh the page.

### Customizing the Viewer

Edit `docs/index.html` directly - it's a static file you can customize:
- Change colors, fonts, layout
- Add custom buttons or features
- Modify camera angles
- Your changes won't be overwritten!

## üìê Units & Measurements

- **Input Units**: 10 units = 1 foot
- **Export**: Converted to meters (standard for 3D)
- **Example**: Wall thickness of 8 units = 8 inches

## üé® Object Types

- `floor_slab` - Floor slabs with configurable thickness
- `room` - Rooms with automatic wall generation
- `wall` - Individual walls with optional sloping tops
- `pillar` - Square pillars/columns
- `door` - Door openings with boolean operations
- `window` - Window openings with sill height
- `staircase` - Staircases with configurable steps
